HTML Tables

HTML <table> Tag

The <table> tag defines an HTML table. An HTML table consists of one <table> element and one or more <tr>, <th>, and <td> elements. The <tr> element defines a table row, the <th> element defines a table header, and the <td> element defines a table cell. An HTML table may also include <caption>, <colgroup>, <thead>, <tfoot></tfoot>, and <tbody> elements.

Most browsers will display the <table>element with the following default values:

HTML colspan Attribute

The colspan attribute defines the number of columns a cell should span.

HTML rowspan Attribute

The rowspan attribute specifies the number of rows a cell should span.
